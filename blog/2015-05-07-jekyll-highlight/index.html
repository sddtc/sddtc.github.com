Jekyll我们都会喜欢的语法高亮

  • 07 May 2015
    • 标签:
  • ruby

jekyll语法高亮之pygments系列：

  1. 下载Pygments Pygments-x.x.tar.gz,解压安装
sudo python setup.py install
  1. 选择一种喜欢的代码高亮样式，我用的native
    通过命令可以查看支持的样式:
>>>from pygments.styles import STYLE_MAP
>>>STYLE_MAP.keys()
  1. 应用在jekyll里:
cd $path/css
pygmentize -S native -f html > pygments.css
  1. 在layout中引用该pygments.css
  2. 修改jekyll的_config.yml
markdown: redcarpet
pygments: true
